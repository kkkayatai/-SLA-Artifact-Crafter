# SL:Arise Artifact Auto-Reroller

An automated, local OCR macro that rerolls your Solo Leveling: Arise artifacts until it hits a perfect 4/4 substat match.

## Requirements

1. **Python 3.10+**: Download from [python.org](https://www.python.org/downloads/). 
   - ⚠️ **Check "Add python.exe to PATH"** during installation!
2. **Tesseract OCR**: Download from [UB-Mannheim](https://github.com/UB-Mannheim/tesseract/wiki). 
   - ⚠️ **Install to the default path**: `C:\Program Files\Tesseract-OCR\tesseract.exe`

## How to Use

1. Double-click **`Run.bat`**. It automatically installs required packages and opens the menu.
2. Type the number for your desired artifact preset and press Enter.
3. **Calibrate (First Time Only):**
   - Hover the top-left of the substats box -> press `F1`.
   - Hover the bottom-right of the substats box -> press `F1`.
   - Hover over the "Readjust Stats" button -> press `F1`.
4. Press **`F2`** to start the auto-reroller!

## Controls
- `F1`: Calibrate screen coordinates (saves automatically).
- `F2`: Start the bot.
- `F3`: Pause/Resume.
- `F5`: Exit.
